# Volatile Tortoiseshell

Desc:
El Pablo, the professor's tortoiseshell cat seems to have tried to contact the crew and send them the mint's digital keys from the professor's workstation. Pablo tried to evade the police's network receivers by hiding the message as much as possible but accidentally hid it from the crew as well. Rio could only retrieve the RAM dump from the connected computer but can't seem to find the message, can you?
